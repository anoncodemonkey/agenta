# Tweeter Agent

A Twitter-like agent implementation built with Node.js.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file and add your configuration:
```
PORT=3000
MONGODB_URI=your_mongodb_uri
```

3. Start the server:
```bash
npm start
```

For development with auto-reload:
```bash
npm run dev
```

## Testing

Run tests with:
```bash
npm test
```
